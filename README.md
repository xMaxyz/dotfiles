My configuration of my Arch/Niri setup.
I initially used Hyprland but recently moved to Niri.
Different themes can be used by running ./theme in the git root. xdg-portal (specifically XDG_CONFIG_HOME) is needed.
It is advised to have $HOME/.local/bin in PATH.
Theme 'dust' inspiration from github.com/af44n/warm_gnome_rice.
'colors' lists most used colors for faster reference. The file itself doesn't configure anything.  
I'm still a newbie when it comes to ricing, so I welcome any suggestions for improvement.
